# RDD Union 
Simple example showing the reading of files as RDDs
and generating the union which is written out to a file.

## Dependencies

- Java 8
- Maven
- Spark
    - version: spark-1.6.0-bin-hadoop2.6
    - get it here: http://spark.apache.org/downloads.html
    
# Build

```bash
cd $project
mvn package
```
